# Установка
$ `apt update`<br>
$ `apt upgrade`<br>
# если не устанавливали пайтон то устанавливйте
$ `apt install python`<br>
$ `apt install python2`<br>
$ `pkg install git`<br>
$ `git clone https://github.com/Lol437/sms.git`<br>
$ `cd spammer`<br>
$ `sh install.sh`<br>
# Запуск
$ `spammer`<br>
